# Deploy

> Source: `docs/deploy/index.mdx`
> Canonical URL: https://sandboxagent.dev/docs/deploy
> Description: Choose where to run the sandbox-agent server.

---
- [Local](/deploy/local) — Run locally for development. The SDK can auto-spawn the server.

- [E2B](/deploy/e2b) — Deploy inside an E2B sandbox with network access.

- [Daytona](/deploy/daytona) — Run in a Daytona workspace with port forwarding.

- [Docker](/deploy/docker) — Build and run in a container (development only).
